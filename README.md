# AI-Powered Chatbot

## Objective

This project is a simple AI-powered chatbot that interacts with users and provides responses to common questions.

## Technologies Used

* Python
* Streamlit
* NLP Concepts

## Features

* User-friendly web interface
* Responds to greetings
* Answers basic AI and Python questions
* Interactive chatbot experience

## How to Run

1. Install Streamlit:

   py -m pip install streamlit

2. Run the chatbot:

   py -m streamlit run chatbot.py

3. Open http://localhost:8501
in your browser.

4. Start chatting with the bot.
